# 📖 Fullstack Development <a name="leaderboard"></a>

### Setup

Clone this repository to your desired folder:
```sh
  git clone https://github.com/sagieramos/leaderboard $HOME
```

To run the project, execute the following command:

```sh
  cd $HOME/leaderboard
  npm install
  npm run build
  npm run dev
```
